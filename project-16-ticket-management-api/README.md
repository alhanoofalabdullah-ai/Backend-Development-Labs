# Ticket Management API

A professional Flask backend project focused on managing support tickets using REST API operations.

---

## Overview

This project demonstrates how to build a backend API for support ticket management, including ticket creation, assignment, priority tracking, status updates, and deletion.

---

## Features

- Create support tickets
- View all tickets
- Get ticket by ID
- Assign tickets to agents
- Update ticket priority
- Update ticket status
- Delete tickets
- Clean JSON responses
- Input validation

---

## Project Structure

- app.py → Flask application
- requirements.txt → Project dependencies
- README.md → Project documentation

## Technologies Used

- Python

- CRUD Operations
- Ticket Management
