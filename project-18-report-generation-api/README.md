# Report Generation API

A professional Flask backend project focused on generating reports using REST API operations.

---

## Overview

This project demonstrates how to build a backend API for generating business reports such as sales reports, user reports, and system activity summaries.

---

## Features

- Create reports
- View all reports
- Get report by ID
- Generate report status tracking
- Mark reports as completed
- Delete reports
- Clean JSON responses
- Input validation

---

## Project Structure

- app.py → Flask application
- requirements.txt → Project dependencies
- README.md → Project documentation

## Technologies Used

- Python
- Flask
- REST API
- CRUD Operations
- Report Management
