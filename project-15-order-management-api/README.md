# Order Management API

A professional Flask backend project focused on managing customer orders using REST API operations.

---

## Overview

This project demonstrates how to build a backend API for order management, including order creation, status updates, order tracking, and deletion.

---

## Features

- Create orders
- View all orders
- Get order by ID
- Update order details
- Update order status
- Delete orders
- Clean JSON responses
- Input validation

---

## Project Structure

- app.py → Flask application
- requirements.txt → Project dependencies
- README.md → Project documentation

---

## Technologies Used

- Python
- Flask
- REST API
- CRUD Operations
- Order Management
