# System Health Monitoring API

A professional Flask backend project focused on monitoring system health and service status using REST API operations.

---

## Overview

This project demonstrates how to build a backend API for system monitoring, including server status, CPU usage, memory usage, disk health, and service availability tracking.

---

## Features

- Check system health status
- Monitor CPU usage
- Monitor memory usage
- Monitor disk usage
- Track service availability
- View health reports
- Clean JSON responses
- Input validation

---

## Project Structure

- app.py → Flask application
- requirements.txt → Project dependencies
- README.md → Project documentation

## Technologies Used

- Python

- System Monitoring
- Health Reporting
