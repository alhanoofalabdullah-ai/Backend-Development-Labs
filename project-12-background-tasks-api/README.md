# Background Tasks API

A professional Flask backend project focused on running background tasks and non-blocking operations.

---

## Overview

This project demonstrates how to start long-running tasks in the background while returning an immediate API response.

---

## Features

- Start background tasks
- Non-blocking API response
- Simulate long-running operations
- Task status tracking
- Clean JSON responses

---

## Project Structure

- app.py → Flask application
- requirements.txt → Project dependencies
- README.md → Project documentation

## Technologies Used

- Python
- Flask
- Threading
- REST API
- Background Processing
