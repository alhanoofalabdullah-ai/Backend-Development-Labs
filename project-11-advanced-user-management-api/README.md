# Advanced User Management API

A professional backend API project focused on managing users, roles, and account status using Flask.

---

## Overview

This project extends basic user management by adding role assignment, account activation status, user search, and structured API responses.

---

## Features

- Create users
- View all users
- Search users by username
- Update user role
- Activate or deactivate accounts
- Delete users
- JSON API responses

---

## Project Structure

- app.py → Main Flask application
- requirements.txt → Project dependencies
- README.md → Project documentation

---

## Technologies Used

- Python
- Flask
- REST API
- JSON
- Backend Development
