# Notification Management API

A professional Flask backend project focused on managing system notifications using REST API operations.

---

## Overview

This project demonstrates how to build a backend API for notification management, including creating notifications, marking them as read, filtering unread notifications, and deleting notifications.

---

## Features

- Create notifications
- View all notifications
- Get notification by ID
- Filter unread notifications
- Mark notifications as read
- Delete notifications
- Clean JSON responses
- Input validation
