# Product Inventory API

A professional Flask backend project focused on managing product inventory using REST API operations.

---

## Overview

This project demonstrates how to build a backend API for product inventory management, including product creation, updates, stock tracking, and deletion.

---

## Features

- Add products
- View all products
- Get product by ID
- Update product details
- Update stock quantity
- Delete products
- Clean JSON responses
- Input validation

---

## Project Structure

- app.py → Flask application
- requirements.txt → Project dependencies
- README.md → Project documentation

---

## Technologies Used

- Python
- Flask
- REST API
- CRUD Operations
- Inventory Management
