# Audit Log API

A professional Flask backend project focused on managing audit logs and tracking system activities using REST API operations.

---

## Overview

This project demonstrates how to build a backend API for audit logging, including activity tracking, event history, user actions monitoring, and log filtering.

---

## Features

- Create audit logs
- View all audit logs
- Get log by ID
- Filter logs by user
- Filter logs by action
- Delete logs
- Clean JSON responses
- Input validation

---
