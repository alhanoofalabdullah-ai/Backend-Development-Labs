# Advanced User Management API

A professional Flask backend project focused on advanced user management with full CRUD operations and search functionality.

---

## Overview

This project extends basic user management by adding advanced CRUD operations, search functionality, validation, and clean API responses.

---

## Features

- Create users
- View all users
- Get single user details
- Update users
- Delete users
- Search users by name
- Search users by email
- Input validation
- Clean JSON responses

---

## Project Structure

- app.py → Flask application
- requirements.txt → Project dependencies
- README.md → Project documentation

---

## Technologies Used

- Python
- Flask
- REST API
- CRUD Operations
- Search & Filtering
